# HomeStock
 StockMate

 HomeStock is a smart web application designed to simplify home inventory and grocery management. It allows households to collaboratively track items, monitor expiry dates, and receive low-stock alerts. The app auto-generates shopping lists using AI-driven recommendations and compares grocery prices through web scraping and API integrations. With detailed reports and visual insights on spending, usage, and waste, HomeStock empowers users to make smarter, budget-friendly household decisions.
